This site is a knowledge-sharing portal for use in school. It allows easy access to learning resources, information sharing, and community.




Self-Exploration

If you are having trouble finding a topic for independent exploration, this tool is for you.
This tool allows you to search past studies of our seniors.You can also use it to search for research and papers by teachers.

waiting time

If you're suffering from cafeteria crowding, this tool is for you!
This tool allows you to search for current dining hall wait times. It also provides access to historical data so you can use it as a reference when using the cafeteria.

Question Box

If you are having trouble with something you don't understand, this tool is for you!
This tool allows technical college students to ask each other questions. You can also use chat gpt.

ploblem generation

If you have a small number of problems in specialized subjects, this tool is for you.
This tool can automatically create questions at random. It is easy to use by selecting the subject and range you wish to study.

deta

If you are having trouble finding a topic for independent exploration, this tool is for you.
This tool allows you to share data collected by others. You may make unexpected discoveries from a lot of data.

learning URL

If you are interested in learning to use the Internet, this tool is for you.
This tool allows you to search for videos and websites recommended by technical college students according to a theme. You can also add your own recommended sites, and it will automatically evolve more and more. Let's make it more user-friendly with everyone's help!
